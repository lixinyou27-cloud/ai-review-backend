AI复盘工具 - 使用指南

====================================================================
如何运行项目
====================================================================

最简单的方法是直接双击运行项目文件夹中的 `run_project.bat` 文件。
这个脚本会自动帮您完成以下步骤：
1. 安装项目依赖
2. 测试OpenAI API连接
3. 启动后端服务

后端服务启动后，会在 http://localhost:5000 运行。
然后您只需要在浏览器中打开项目文件夹中的 `index.html` 文件，就可以开始使用AI复盘工具了。


====================================================================
手动运行方式（如果一键脚本有问题）
====================================================================

1. 打开命令提示符（按Win+R，输入cmd，按回车）

2. 切换到项目目录：
   cd d:\1-软件\Ai编程

3. 安装依赖：
   d:\1-软件\python\python.exe -m pip install -r requirements.txt

4. 测试API连接（可选但推荐）：
   d:\1-软件\python\python.exe test_with_full_path.py

5. 启动后端服务：
   d:\1-软件\python\python.exe app.py

6. 打开浏览器，访问项目文件夹中的index.html文件


====================================================================
使用AI复盘工具
====================================================================

1. 在前端页面上，填写您的复盘内容
2. 选择相关标签（可选）
3. 点击"分析"按钮
4. 系统会调用后端API进行AI分析，或者在API不可用时使用模拟数据
5. 查看分析结果，包括：
   - 内容拆解
   - 核心洞察
   - 行动策略
   - 学习资源推荐
   - 潜在天赋发现
   - 商业机会分析
   - 明日待办清单


====================================================================
常见问题解决
====================================================================

Q: 运行脚本时提示Python路径不存在
A: 请用记事本打开run_project.bat文件，修改PYTHON_PATH变量为您实际的Python安装路径

Q: API测试失败
A: 可能的原因包括：
   - API密钥不正确，请检查.env文件中的配置
   - OpenAI账户余额不足
   - 网络连接问题
   - OpenAI服务暂时不可用

Q: 如何停止后端服务
A: 在运行服务的命令提示符窗口中，按Ctrl+C组合键即可停止服务

Q: 前端页面可以正常打开，但AI分析功能不工作
A: 请确保后端服务正在运行，检查浏览器控制台是否有错误信息


====================================================================
注意事项
====================================================================

- 请妥善保管您的OpenAI API密钥，不要泄露给他人
- 后端服务仅在本地运行，不会被外部网络访问
- 当后端服务不可用时，前端会自动切换到模拟数据模式
- 定期更新您的依赖包以获取最新功能和安全修复

祝您使用愉快！
